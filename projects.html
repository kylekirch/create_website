Fishing Calculator

Sept 2022 - Dec 2022

A data-driven “fishing score” application which assigned weighted values to wind speed, precipitation, humidity, temperature delta, etc. to calculate an intuitive “fishing score”, indicative of traditionally optimal fishing conditions

Key Features

  • User-friendly interface, enabling anglers to input exact coordinates and date parameters to generate personalized fishing scores
  • A graph showing fishing scores of any given location for the next 7 days
  • Utilizes up-to-date weather information directly from the NOAA's public API

Technologies & Skills

Django Python JavaScript Data Visualization NOAA Public API Real-time Updates User Friendly GUI
